# Product

## Register

product

## Users

Lumina is for local developers working in desktop project directories. They need to inspect Git state, prepare commits with AI assistance, and later control project run/build tasks from the same workspace.

## Product Purpose

Lumina turns a selected project directory into a focused desktop workbench. Success means a developer can understand the current project state, choose the right files, generate or edit a commit message, and run the necessary project actions without losing context.

## Brand Personality

Restrained, professional, efficient.

## Anti-references

Lumina should not feel like a marketing page, a decorative dashboard, or a novelty AI interface. It should avoid heavy ornamentation, oversized hero patterns, gratuitous motion, and layouts that trade information density for visual flair.

## Design Principles

- Keep the selected project as the stable context across tools.
- Make the next action obvious before exposing secondary controls.
- Preserve developer-tool density while separating unrelated decisions.
- Use AI as an assistive layer, not as the center of the workflow.
- Prefer familiar desktop workbench patterns over invented navigation.

## Accessibility & Inclusion

Treat Lumina as a professional desktop tool: maintain readable contrast, visible focus states, keyboard-reachable primary actions, and status text that is not conveyed by color alone.
